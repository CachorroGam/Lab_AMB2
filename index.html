CILINDRO DE GAS

PROYECTO DE LA APLICACIÓN DE LA DERIVADA, OPTIMIZACION DE MATERIALES Y COSTO DE UN CILINDRO DE GAS.

Información del proyecto

Objetivo General: Presentar y visualizar de manera interactiva el modelo 3D del proyecto "Cilindro de Gas", permitiendo explorar, manipular y comprender su estructura de manera dinámica.

Objetivos Específicos:

  • Desarrollar un visor 3D que permita rotar, acercar, alejar y cambiar el color del modelo.
  • Garantizar la compatibilidad y accesibilidad del sitio en diferentes dispositivos y navegadores.
  • Evaluar la experiencia para asegurar que la navegación y la interacción con el modelo sean intuitivas.

En este proyecto buscamos ofrecer una experiencia interactiva diferente, desarrollando una página web que permite visualizar y explorar en 3D el modelo del proyecto "Cilindro de Gas". ¡Esperamos que disfrute navegando y descubriendo cada detalle de nuestro trabajo! :D

Visor 3D

Instrucciones de uso del modelo 3D:
En computadora: arrastrar con el mouse para rotar, usar la rueda del scroll para hacer zoom y botón derecho para mover.
En teléfono o tablet: pellizcar para hacer zoom, arrastrar con el dedo para rotar y tocar con dos dedos para mover.
